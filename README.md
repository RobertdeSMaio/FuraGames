# 🎮 FuraGames - Gestão de Análises de Jogos

O **FuraGames** é uma aplicação Fullstack desenvolvida para gamers que desejam organizar as suas experiências, partilhar análises detalhadas e interagir com comunidades através de grupos. 

A plataforma permite que os utilizadores registem o seu progresso, avaliem jogos com base em horas jogadas e troquem feedback através de comentários nas análise.

## 🚀 Tecnologias Utilizadas

- **Frontend:** [Next.js](https://nextjs.org/) (React) com Turbopack.
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/).
- **ORM:** [Prisma](https://www.prisma.io/).
- **Base de Dados:** [PostgreSQL (Neon.tech)](https://neon.tech/).
- **Deploy:** [Vercel](https://vercel.com/).
- **Ícones:** [Lucide React](https://lucide.dev/).

## 🛠️ Funcionalidades

- **Autenticação de Utilizadores:** Registo e login de perfis personalizados.
- **Reviews de Jogos:** Criação de análises com título, plataforma, género, rating, horas jogadas e listas de prós/contras.
- **Grupos de Comunidade:** Criação de grupos e convites via código único para discussões privadas.
- **Sistema de Comentários:** Interação direta em cada review postada.
- **Status Dinâmico:** Organização por estados: *Playing, Completed, Dropped, On-hold* ou *Want-to-play*.
